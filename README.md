# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes one year of pizza sales data to uncover business insights related to sales performance, customer ordering behavior, product popularity, and revenue trends.

The analysis was performed using SQL for data extraction and Microsoft Excel for KPI reporting and dashboard creation.

---

## 🎯 Objectives

- Analyze overall sales performance
- Identify top and bottom selling pizzas
- Understand customer ordering patterns
- Analyze peak order days and hours
- Compare pizza categories and sizes
- Generate business insights for decision making

---

## 📊 Key Performance Indicators (KPIs)

- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Daily Customers
- Average Pizza Price
- Average Pizzas per Order
- Total Pizza Varieties

---

## 📈 Business Insights

- Identified best and worst selling pizzas.
- Analyzed customer preferences across pizza categories and sizes.
- Determined peak ordering hours and days.
- Evaluated monthly revenue trends.
- Examined ingredient demand and popularity.
